Introduction
Welcome to the Property Purchase Simulation program. This application is designed to help potential property buyers understand the financial implications of purchasing a property, including calculations for stamp duty, mortgage options, and monthly housing expenses. The program is particularly oriented towards the context of Hong Kong's real estate market, allowing users to input various parameters to simulate their buying experience.

Features
Stamp Duty Calculation: Based on the property price, the program calculates the applicable stamp duty.
Mortgage Calculation:
Calculates the amount of first mortgage based on user-defined percentage.
Determines if a second mortgage is necessary if the mortgage exceeds 70%.
Allows calculation of monthly repayment for loans with user-defined terms.
User-Friendly Input: The program prompts users for all necessary inputs in a clear manner.
Output Summary: Displays detailed results, including total expenses and necessary financial commitments.
How It Works
1. User Input
When the program starts, it will require the following inputs from the user:

Property Price: The overall cost of the property being purchased.
Mortgage Percentage: The percentage of the property price that you plan to finance through a mortgage.
If the mortgage percentage is above 70%, additional inputs will be prompted:
Monthly Income: Your monthly income for repayment calculation.
Repayment Period: The number of years over which you intend to repay the mortgage.
Interest Rate: The annual interest rate applicable to the mortgage.
2. Calculations Performed
After capturing user inputs, the program performs several calculations:

Stamp Duty: Calculated based on standard rates relative to the property price.
Mortgage Amounts:
First Mortgage: The initial amount borrowed.
Second Mortgage (if applicable): The amount required if the first mortgage does not cover the property's cost.
Monthly Expenses:
If a second mortgage is necessary, the program calculates the monthly repayment for it.
The program also computes total monthly housing expenses, combining repayments for the first and any second mortgage.
3. Output
The program will display the following results:

Stamp duty owed.
Amount of the first mortgage.
Amount required for a second mortgage (if applicable).
Monthly housing expense calculated based on mortgage terms.
Example Usage
To run the program, ensure you have Python installed. Save the script into a .py file (e.g., property_purchase.py) and execute the following command in your terminal:

bash
python property_purchase.py

Follow the on-screen prompts to enter the required information. You will receive an output summarizing your financial commitments related to the property purchase.

By providing a clear breakdown of financial factors, the program empowers users to make informed decisions on home buying and financing.
